Embedded DSP Streamer

A high-performance DSP data streaming system using STM32H7, MAX10 FPGA and i.MX8.

üöÄ Project Overview

The Embedded DSP Streamer is a modular embedded pipeline for high-speed signal acquisition and streaming. It combines:

  • STM32H745 (FMC + MDMA data acquisition)
  • Intel MAX10 FPGA (DSP, NCO, custom logic)
  • i.MX8 embedded Linux SBC (UDP streaming & processing)

The goal is to build a reliable, low-latency, real-time signal chain for experiments, DSP prototyping and hardware learning.
